Utterances 소개#

Utterances는 GitHub Issue 기반으로 Comment를 작성할 수 있게 해줍니다.

Utterances의 장점

  • GitHub Issue 기반이기 때문에 블로그 플랫폼을 이전해도 기존 Comment을 그대로 가져갈 수 있음
  • 개발 블로그 라면 Github 계정으로 댓글을 작성할 수 있기 때문에 사용성이 용이함
  • 뭔가 있어 보임, 유명한 블로그는 이거 쓰는 거 같음
