인텔리제이 Github Code Review

2020/07/18

인텔리제이를 사용하면 Github의 Pull Request를 확인하고, 코드 리뷰를 진행할 수 있습니다. Github 기반 PR 관리는 Github로 프로젝트 관리하기 Part1을 참고하세요.

1. PR 목록 확인#

인텔리제이 상단의 Actions 메뉴에서 View Pullrequests를 선택하면, 현재 열려있는(Pull Request 상태: state: open) PR 목록을 한눈에 확인할 수 있습니다.

2. 코드 리뷰 및 Diff 확인#

PR 리스트에서 코드 리뷰 대상 파일들이 우측에 표시됩니다. 예를 들어, User.java 파일을 클릭하면 변경된 내용을 Diff 형식으로 자세히 확인할 수 있습니다.

3. 코멘트 추가#

Diff 뷰에서 특정 코드 라인에 코멘트를 추가하려면, 해당 라인 왼쪽에 있는 + 버튼을 클릭합니다. 작성한 코멘트는 Github와 인텔리제이 모두에서 확인할 수 있으며, 다른 리뷰어의 피드백도 달 수 있습니다.

4. PR Merge#

인텔리제이 내에서 Merge 버튼을 클릭하여 Pull Request를 바로 머지할 수 있습니다. 실제 머지 결과는 인텔리제이 PR & Code Review #24에서 확인할 수 있습니다.

5. 검색 및 필터링#

PR 목록 상단의 검색 기능을 사용하면, status, author, assignee 등의 조건으로 원하는 PR을 빠르게 필터링할 수 있습니다.
